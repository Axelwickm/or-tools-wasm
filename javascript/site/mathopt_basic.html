MathOpt - basic model sample

import { MathOpt, initMathOpt } from 'or-tools-wasm';

MathOpt.setWorkerBridgeEnabled(true);
await initMathOpt();

const model = MathOpt.Model('Basic MathOpt example');
const x = model.addVariable({ lowerBound: 0, upperBound: 1, name: 'x' });
const y = model.addVariable({ lowerBound: 0, upperBound: 2.5, name: 'y' });

model.addLinearConstraint({ upperBound: 1.5, terms: [
  { variable: x, coefficient: 1 },
  { variable: y, coefficient: 1 },
]});
model.maximize([
  { variable: x, coefficient: 2 },
  { variable: y, coefficient: 1 },
]);

const result = await MathOpt.solve(model, {
  solverType: MathOpt.SolverType.GLOP,
  threads: 4,
});
console.log(result.objectiveValue, result.variableValues);

Adapted from ortools/math_opt/samples/python/basic_example.py. The browser demo solves the same two-variable linear model with GLOP.
