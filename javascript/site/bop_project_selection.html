BOP - binary project selection

Problem: choose the most valuable set of binary projects without exceeding a budget or overloading one shared team.

Show code 
import { initMPSolver, MPSolver, setWorkerBridgeEnabled } from 'or-tools-wasm/mp-solver';

setWorkerBridgeEnabled(true);
await initMPSolver();

const solver = MPSolver.CreateSolver('BOP');
if (!solver) throw new Error('BOP backend is unavailable');

const analytics = solver.BoolVar('analytics');
const dashboard = solver.BoolVar('dashboard');
const alerts = solver.BoolVar('alerts');

const budget = solver.Constraint(-solver.infinity(), 9, 'budget');
budget.SetCoefficient(analytics, 6);
budget.SetCoefficient(dashboard, 4);
budget.SetCoefficient(alerts, 3);

const sharedTeam = solver.Constraint(-solver.infinity(), 1, 'shared_design_team');
sharedTeam.SetCoefficient(dashboard, 1);
sharedTeam.SetCoefficient(alerts, 1);

const objective = solver.Objective();
objective.SetCoefficient(analytics, 8);
objective.SetCoefficient(dashboard, 6);
objective.SetCoefficient(alerts, 5);
objective.SetMaximization();

const status = await solver.Solve();
if (status !== MPSolver.OPTIMAL) throw new Error(`expected OPTIMAL, got ${status}`);

console.log(objective.Value(), {
  analytics: analytics.solution_value(),
  dashboard: dashboard.solution_value(),
  alerts: alerts.solution_value(),
});

BOP solves pure Boolean and integer MPSolver models. This example uses binary variables, a budget constraint, and a shared-team conflict constraint.

Run the solver to view the selected projects.
