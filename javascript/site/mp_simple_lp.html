MPSolver - frontend sample

Problem: solve one small linear model through MPSolver while switching between available LP and integer backends.

Show code 
import { initMPSolver, MPSolver } from 'or-tools-wasm';

await initMPSolver();

const solver = MPSolver.CreateSolver('GLOP');
if (!solver) throw new Error('GLOP is unavailable');

const x = solver.NumVar(0, 1, 'x');
const y = solver.NumVar(0, 1, 'y');
const c0 = solver.Constraint(-solver.infinity(), 1);
c0.SetCoefficient(x, 1);
c0.SetCoefficient(y, 1);

const objective = solver.Objective();
objective.SetCoefficient(x, 2);
objective.SetCoefficient(y, 1);
objective.SetMaximization();

await solver.SolveWithProto();
console.log(x.solution_value(), y.solution_value());

MPSolver routes one tiny linear model through selectable backends. GLOP, CLP, and GLPK_LP solve it as a continuous LP; SAT solves the same model with integral variables.

Run the solver to view the solution.
